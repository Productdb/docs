# ProductDBParticipantsToggle

A button which toggles visibility of participants.

When clicked it emits a `ProductDBStateUpdate` event with the data:

```ts
{
  activeSidebar: boolean;
  sidebar: 'participants' | 'none';
}
```

```jsx live
<Center>
  <ProductDBParticipantsToggle size="sm" />
  <ProductDBParticipantsToggle size="lg" />
  <ProductDBParticipantsToggle variant="horizontal" size="sm" />
</Center>
```
