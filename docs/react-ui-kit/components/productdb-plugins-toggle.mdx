# ProductDBPluginsToggle

A button which toggles visibility of plugins.

When clicked it emits a `dyteStateUpdate` event with the data:

```ts
{
  activeSidebar: boolean;
  sidebar: 'plugins' | 'none';
}
```

```jsx live
<Center>
  <ProductDBPluginsToggle size="sm" meeting={meeting} />
  <ProductDBPluginsToggle size="lg" meeting={meeting} />
  <ProductDBPluginsToggle variant="horizontal" size="sm" meeting={meeting} />
</Center>
```
