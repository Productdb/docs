# ProductDBSidebar

A component which handles the sidebar and
you can customize which sections you want, and which section you want as the default.

```jsx live
<Center>
  <ProductDBSidebar
    defaultSection="participants"
    meeting={meeting}
    style={{ height: '480px' }}
  />
</Center>
```

To see a mobile sidebar:

```jsx live
<div style={{ position: 'relative', height: '600px' }}>
  <ProductDBSidebar
    view="full-screen"
    defaultSection="participants"
    meeting={meeting}
    style={{ maxWidth: '360px', margin: 'auto' }}
  />
</div>
```
