# ProductDBSettingsToggle

A button which toggles visibility of chat.

When clicked it emits a `ProductDBStateUpdate` event with the data:

```ts
{
  activeSettings: boolean;
}
```

```jsx live
<Center>
  <ProductDBSettingsToggle size="sm" />
  <ProductDBSettingsToggle size="lg" />
  <ProductDBSettingsToggle variant="horizontal" size="sm" />
</Center>
```
