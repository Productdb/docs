# ProductDBParticipantTile

import CodeRunner from '@site/src/components/CodeRunner/CodeRunner';

A component which plays a participants video and allows for placement
of components like `productdb-name-tag`, `productdb-audio-visualizer` or any other component.

<CodeRunner 
hide={[{start:1, end: 3}]}
minHeight='300px'
file={`
import { ProductDBParticipantTile, ProductDBNameTag, ProductDBAudioVisualizer } from "@dytesdk/react-ui-kit";
import { useDyteMeeting } from "@dytesdk/react-web-core";

export default function MyMeetingUI() {
  const { meeting } = useDyteMeeting();
  return (
    <ProductDBParticipantTile participant={meeting.self}>
      <ProductDBNameTag participant={meeting.self}>
        <ProductDBAudioVisualizer slot="start" />
      </ProductDBNameTag>
    </ProductDBParticipantTile>
  );
}
`} />

You can change the `nameTagPosition` to any of the supported values
and change the placement of audio-visualizer in name-tag as well.
