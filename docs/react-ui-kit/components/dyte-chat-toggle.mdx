# ProductDBChatToggle

A button which toggles visibility of chat.

You need to pass the `meeting` object to it to see the unread messages count badge.

When clicked it emits a `ProductDBStateUpdate` event with the data:

```ts
{
  activeSidebar: boolean;
  sidebar: 'chat' | 'none';
}
```

```jsx live
<Center>
  <ProductDBChatToggle size="sm" meeting={meeting} />
  <ProductDBChatToggle size="lg" meeting={meeting} />
  <ProductDBChatToggle variant="horizontal" size="sm" meeting={meeting} />
</Center>
```
