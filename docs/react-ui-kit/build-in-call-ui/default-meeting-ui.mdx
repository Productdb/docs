ProductDB provides, `ProductDBMeeting` an all encompassing component that internally handles everything from showing a pre-call UI to in-call UI and post-call screen.

```tsx
<ProductDBMeeting meeting={meeting} showSetupScreen={true} />
```

This component contains pre-call, in-call UI as well post-call UIs.

Following code shows a basic split of these UIs from the `ProductDBMeeting` component.
