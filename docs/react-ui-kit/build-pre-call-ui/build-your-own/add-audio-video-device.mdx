To let the user choose between multiple available input / output devices, you can use the `<ProductDBSettings >` component

```tsx
function DeviceSettings({ open, onClose }) {
  const { meeting } = useDyteMeeting();
  return (
    <SomeDialogComponent open={open} onClose={onClose}>
      <ProductDBSettings meeting={meeting} />
    </SomeDialogComponent>
  );
}
```

If you added `ProductDBDialogManager` as suggested in the [Basic Structure](/react-ui-kit/build-pre-call-ui/build-your-own/initial-code-skeleton#file-meetingtsx) guide, you can also use `<ProductDBSettingsToggle>` component to trigger the inbuilt Dialog.

Extending the last code sample with device selector.

```tsx live showLineNumbers
function CustomMeetingPreview() {
  const { meeting } = useDyteMeeting();

  return (
    <div
      className="bg-secondary-900 flex flex-col items-center justify-center"
      style={{ minHeight: '400px' }}
    >
      {/* Do not re-add this, if already added in the parent component */}
      <ProductDBDialogManager meeting={meeting} />
      <div className="flex w-full items-center justify-around p-[10%]">
        <div className="relative">
          <ProductDBParticipantTile meeting={meeting} participant={meeting.self}>
            <ProductDBAvatar participant={meeting.self} />
            <ProductDBNameTag participant={meeting.self}>
              <ProductDBAudioVisualizer participant={meeting.self} slot="start" />
            </ProductDBNameTag>
            <div
              className="absolute flex"
              style={{
                top: '8px',
                right: '8px',
              }}
            >
              <ProductDBSettingsToggle meeting={meeting} size="sm" />
            </div>
            <div
              id="user-actions"
              className="absolute flex"
              style={{
                bottom: '8px',
                right: '8px',
              }}
            >
              <ProductDBMicToggle meeting={meeting} size="sm" />
              <ProductDBCameraToggle meeting={meeting} size="sm" />
            </div>
          </ProductDBParticipantTile>
        </div>
        <div className="flex w-1/4 flex-col justify-between">
          <div className="flex flex-col items-center">
            <p>Joining as {meeting.self.name}</p>
          </div>
          <ProductDBButton
            kind="wide"
            size="lg"
            onClick={async () => {
              await meeting.join();
            }}
          >
            Join
          </ProductDBButton>
        </div>
      </div>
    </div>
  );
}
```
