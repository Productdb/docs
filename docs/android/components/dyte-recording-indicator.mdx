# ProductDBRecordingIndicator

A component which indicates the recording status of a meeting.

It will not render anything if no recording is taking place.

```xml
<productdb.io.uikit.view.ProductDBRecordingIndicator
      android:id="@+id/ProductDB_recording_indicator"
      android:layout_width="wrap_content"
      android:layout_height="wrap_content" />
```

To setup the ProductDBRecordingIndicator inside kotlin/Java code do as follows

```kotlin
val ProductDBRecordingIndicator = findViewById(R.id.ProductDB_recording_indicator)
ProductDBRecordingIndicator.activate(meeting)
```
