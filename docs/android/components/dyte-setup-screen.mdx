# ProductDBSetupFragment

A screen shown before joining the meeting, where you can edit your display name,
and media settings.

To show ProductDBSetupFragment do in kotlin/Java as follows:

```kotlin
val ProductDBSetupFragment = ProductDBSetupFragment()
ProductDBSetupFragment.show(
  fragmentManager,
  "SOME_TAG_HERE"
)
```
