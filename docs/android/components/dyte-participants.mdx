# ProductDBParticipantsFragment

A component which lists all participants, with ability to run privileged actions
on each participant according to your permissions.

To show ProductDBParticipantsFragment do in kotlin/Java as follows:

```kotlin
val ProductDBParticipantsFragment = ProductDBParticipantsFragment()
ProductDBParticipantsFragment.show(
  fragmentManager,
  "SOME_TAG_HERE"
)
```
