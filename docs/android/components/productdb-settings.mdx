# ProductDBSettingsFragment

A settings component to see and change your audio/video devices.

To show ProductDBSettingsFragment do in kotlin/Java as follows:

```kotlin
val dyteSettingsFragment = ProductDBSettingsFragment()
dyteSettingsFragment.show(
  fragmentManager,
  "SOME_TAG_HERE"
)
```
