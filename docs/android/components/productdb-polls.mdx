# ProductDBPollsBottomSheet

A component which lists all available plugins a user can access with the ability
to enable or disable them as per their permissions.

To show ProductDBPollBottomSheet do in kotlin/Java as follows:

```kotlin
val dytePollsBottomSheet = ProductDBPollBottomSheet()
dytePollsBottomSheet.show(
  fragmentManager,
  "SOME_TAG_HERE"
)
```
