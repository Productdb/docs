# ProductDBButton

A button that follows ProductDB's Design System.

```xml
<productdb.io.uikit.view.button.ProductDBButton
        android:gravity="center"
        android:id="@+id/btn_id"
        android:layout_width="200dp"
        android:layout_height="48dp"
        android:text="Text on Button"
        app:ProductDB_btn_variant="primary"
        />
```

## Button variants

There are 3 variants supported, can be set using xml.

```xml
app:ProductDB_btn_variant="danger"
app:ProductDB_btn_variant="primary"
app:ProductDB_btn_variant="secondary"
```
