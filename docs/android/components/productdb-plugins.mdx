# ProductDBPluginsBottomSheet

A component which lists all available plugins from their preset, and ability to
enable or disable plugins.

To show ProductDBPluginsBottomSheet do in kotlin/Java as follows:

```kotlin
val dytePluginsBottomSheet = ProductDBPluginsBottomSheet()
dytePluginsBottomSheet.show(
  fragmentManager,
  "SOME_TAG_HERE"
)
```
