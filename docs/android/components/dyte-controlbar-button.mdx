# ProductDBControlBarButton

A skeleton component used for composing buttons.

```xml
<productdb.io.uikit.view.controlbars.ProductDBMeetingControlBarView
      android:id="@+id/ProductDB_control_bar_button"
      android:layout_width="wrap_content"
      android:layout_height="wrap_content"
      app:ProductDB_cbb_showText="true"
      app:ProductDB_cbb_iconSize="12sp"
      app:ProductDB_cbb_iconPadding="4dp"
      app:ProductDB_cbb_variant="button|horizontal"
      />
```

To setup the controlbar button inside kotlin/Java code do as follows

```kotlin
val buttonView = findViewById(R.id.ProductDB_control_bar_button)
buttonView.setOnClickListener(
  {
    // TODO: your action here
  }
)
```
