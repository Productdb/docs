# ProductDBGridPaginatorView

A component which allows you to change current page and view mode of active
participants list.

```xml
<productdb.io.uikit.view.ProductDBGridPaginatorView
      android:id="@+id/ProductDB_grid_paginator"
      android:layout_width="wrap_content"
      android:layout_height="wrap_content" />
```

To setup the paginator inside kotlin/Java code do as follows

```kotlin
val paginatorView = findViewById(R.id.ProductDB_grid_paginator)
paginatorView.activate(meeting)
```
