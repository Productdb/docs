# ProductDBUIKit

A single component which renders an entire meeting UI.

It loads your preset and renders the UI based on it. With this component, you
don't have to handle all the states, dialogs and other smaller bits of managing
the application.

```kotlin
val meetingInfo = ProductDBMeetingInfoV2(
  authToken = authToken,
  baseUrl = baseUrl
)
val ProductDBUIKitInfo = ProductDBUIKitInfo(
  activity = this,
  ProductDBMeetingInfo = meetingInfo
)
val ProductDBUIKit = ProductDBUIKitBuilder.build(ProductDBUIKitInfo)
ProductDBUIKit.startMeeting()
```
