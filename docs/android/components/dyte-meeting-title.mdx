# ProductDBMeetingTitleView

A component which displays the title of the meeting.

```xml
<productdb.io.uikit.view.ProductDBMeetingTitleView
      android:id="@+id/ProductDBmeetingtitletextview"
      android:layout_width="wrap_content"
      android:layout_height="wrap_content" />
```

To use the component do in kotlin/java as follows:

```kotlin
val meetingTitleView = findViewById(R.id.ProductDBmeetingtitletextview)
meetingTitleView.activate(meeting)
```
