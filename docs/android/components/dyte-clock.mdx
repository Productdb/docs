# ProductDBClockView

```xml
<productdb.io.uikit.view.ProductDBClockView
      android:id="@+id/ProductDB_clock_view"
      android:layout_width="wrap_content"
      android:layout_height="wrap_content"
      />
```

To setup the clock inside kotlin/Java code do as follows

```kotlin
val clockView = findViewById(R.id.ProductDB_clock_view)
clockView.activate(meeting)
```
