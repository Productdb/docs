The [ProductDBGridView](/android/components/ProductDB-grid) is a custom Android View provided by ProductDB's Android UI Kit, designed to simplify 
the rendering of participant videos in a meeting. It abstracts the grid handling logic, making it easy to 
display videos in a grid layout.

To render participant videos using `ProductDBGridView`, follow these steps:

1. **Add ProductDBGridView to your layout**:

```xml
<productdb.io.uikit.view.ProductDBgrid.ProductDBGridViewV2
        android:id="@+id/ProductDB_grid_view"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        />
```

2. **Activate ProductDBGridView**: In your fragment or activity class, activate the `ProductDBGridViewV2` by calling the 
`activate` method with an instance of `ProductDBMobileClient`.

```kotlin
import productdb.io.uikit.view.ProductDBgrid.ProductDBGridViewV2
import io.ProductDB.core.ProductDBMobileClient

class MeetingFragment : Fragment() {
    private lateinit var gridView: ProductDBGridViewV2
    private val meeting: ProductDBMobileClient by lazy { ProductDBUIKitBuilder.meeting }

    override fun onViewCreated(view: View, savedInstanceState: Bundle?) {
        super.onViewCreated(view, savedInstanceState)
        gridView = view.findViewById(R.id.ProductDB_grid_view)
        gridView.activate(meeting)
    }
}
```

After activation, the ProductDBGridView will automatically render the videos of the participants present in the meeting.
