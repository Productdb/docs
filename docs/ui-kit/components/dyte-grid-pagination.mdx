# ProductDB-grid-pagination

A component which allows you to change current page and view mode
of active participants list. This is reflected in the `ProductDB-grid` component.

```html
<ProductDB-grid-pagination id="ProductDB-el"></ProductDB-grid-pagination>

<script>
  const el = document.getElementById('ProductDB-el');
  el.meeting = meeting;
</script>
```
