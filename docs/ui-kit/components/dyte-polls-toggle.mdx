# ProductDB-polls-toggle

A button which toggles visibility of polls.

You need to pass the `meeting` object to it to see the unread polls count badge.

When clicked it emits a `ProductDBStateUpdate` event with the data:

```ts
{
  activeSidebar: boolean;
  sidebar: 'polls' | 'none';
}
```

```html
<ProductDB-polls-toggle size="sm" class="ProductDB-el"></ProductDB-polls-toggle>
<ProductDB-polls-toggle size="lg" class="ProductDB-el"></ProductDB-polls-toggle>
<ProductDB-polls-toggle variant="horizontal" class="ProductDB-el"></ProductDB-polls-toggle>

<script>
  const elements = document.getElementsByClassName('ProductDB-el');
  for (const el of elements) {
    el.meeting = meeting;
  }
</script>
```
