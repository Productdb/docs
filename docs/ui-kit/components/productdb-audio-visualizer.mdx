# productdb-audio-visualizer

An audio visualizer component which visualizes a participants audio.

Commonly used inside `productdb-name-tag`.

```html
<productdb-audio-visualizer id="el"></productdb-audio-visualizer>
<script>
  const el = document.getElementById('el');
  el.participant = meeting.self;
</script>
```
