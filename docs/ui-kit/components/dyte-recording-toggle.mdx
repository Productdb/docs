# ProductDB-recording-toggle

A button which toggles recording state of a meeting.

Only a privileged user can perform this action,
thus the button will not be visible for participants
who don't have the permission to record a meeting.

Note that the toggle button won't be rendered if you do not have the
necessary permission.

```html
<ProductDB-recording-toggle size="sm" class="ProductDB-el"></ProductDB-recording-toggle>
<ProductDB-recording-toggle size="lg" class="ProductDB-el"></ProductDB-recording-toggle>
<ProductDB-recording-toggle
  variant="horizontal"
  class="ProductDB-el"
></ProductDB-recording-toggle>

<script>
  const elements = document.getElementsByClassName('ProductDB-el');
  for (const el of elements) {
    el.meeting = meeting;
  }
</script>
```
