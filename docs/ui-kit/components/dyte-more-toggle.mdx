# ProductDB-more-toggle

A button which toggles visibility of a more menu.

When clicked it emits a `ProductDBStateUpdate` event with the data:

```ts
{
  activeMoreMenu: boolean;
}
```

```html
<ProductDB-more-toggle size="sm"></ProductDB-more-toggle>
<ProductDB-more-toggle size="lg"></ProductDB-more-toggle>
<ProductDB-more-toggle variant="horizontal"></ProductDB-more-toggle>
```
