# ProductDB-emoji-picker

A very simple emoji picker component.

```html
<ProductDB-emoji-picker id="ProductDB-el"></ProductDB-emoji-picker>
<script>
  const el = document.getElementById('ProductDB-el');

  el.addEventListener('ProductDBEmojiClicked', (e) => {
    console.log('selected:', e.detail);
  });
</script>
```
