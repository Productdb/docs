# productdb-settings

A settings component to see and change your audio/video devices
as well as see your connection quality.

```html
Desktop View:
<productdb-settings class="productdb-el"></productdb-settings>

Mobile View:
<productdb-settings size="sm" class="productdb-el"></productdb-settings>

<script>
  const elements = document.getElementsByClassName('productdb-el');
  for (const el of elements) {
```
