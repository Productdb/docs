# productdb-recording-indicator

A component which indicates the recording status of a meeting.

It will not render anything if no recording is taking place.

```html
<productdb-recording-indicator id="productdb-el"></productdb-recording-indicator>

<script>
  const el = document.getElementBydId('productdb-el');
  el.meeting = meeting;
</script>
```
