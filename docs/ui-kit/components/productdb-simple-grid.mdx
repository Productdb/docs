# productdb-simple-grid

A grid component which renders only the participants in a simple grid.

```html
<productdb-simple-grid id="productdb-el"></productdb-simple-grid>

<script>
  const el = document.getElementBydId('productdb-el');
  el.participants = [meeting.self];
</script>

<style>
  productdb-simple-grid {
    height: 360px;
    width: 100%;
  }
</style>
```
