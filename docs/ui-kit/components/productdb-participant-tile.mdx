# productdb-participant-tile

A component which plays a participants video and allows for placement
of components like `productdb-name-tag`, `productdb-audio-visualizer` or any other component.

```html
<productdb-participant-tile class="productdb-el">
  <productdb-name-tag class="productdb-el">
    <productdb-audio-visualizer class="productdb-el" slot="start"></productdb-audio-visualizer>
  </productdb-name-tag>
</productdb-participant-tile>

<script>
  const elements = document.getElementsByClassName('productdb-el');
  for (const el of elements) {
    el.participant = meeting.self;
  }
</script>
```

You can change the `name-tag-position` to any of the supported values
and change the placement of audio-visualizer in name-tag as well.

```html
<productdb-participant-tile class="productdb-el" name-tag-position="bottom-center">
  <productdb-name-tag class="productdb-el">
    <productdb-audio-visualizer class="productdb-el" slot="end"></productdb-audio-visualizer>
  </productdb-name-tag>
</productdb-participant-tile>

<script>
  const elements = document.getElementsByClassName('productdb-el');
  for (const el of elements) {
    el.participant = meeting.self;
  }
</script>
```

It also has a few variants.

```html
<productdb-participant-tile class="productdb-el" variant="gradient">
  <productdb-name-tag class="productdb-el">
    <productdb-audio-visualizer class="productdb-el" slot="start"></productdb-audio-visualizer>
  </productdb-name-tag>
</productdb-participant-tile>

<script>
  const elements = document.getElementsByClassName('productdb-el');
  for (const el of elements) {
    el.participant = meeting.self;
  }
</script>
```
