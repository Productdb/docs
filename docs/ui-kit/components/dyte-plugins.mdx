# ProductDB-plugins

A component which lists all available plugins from their preset,
and ability to enable or disable plugins.

```html
<ProductDB-plugins id="ProductDB-el"></ProductDB-plugins>

<script>
  document.getElementById('ProductDB-el').meeting = meeting;
</script>

<style>
  ProductDB-plugins {
    height: 480px;
    width: 100%;
    max-width: 320px;
    background-color: '#000';
  }
</style>
```
