# ProductDB-settings-audio

A component which lets to manage your audio devices and audio preferences.

Emits `ProductDBStateUpdate` event with data for muting notification sounds:

```ts
{
  prefs: {
    muteNotificationSounds: boolean;
  }
}
```

```html
<ProductDB-settings-audio id="ProductDB-el"></ProductDB-settings-audio>

<script>
  const el = document.getElementById('ProductDB-el');
  el.meeting = meeting;
</script>
```
