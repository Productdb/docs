# productdb-screenshare-view

A component which plays a participant's screenshared video.

It also allows for placement of other components similar to `productdb-participant-tile`.

This component will not render anything if the participant hasn't start screensharing.

```html
<productdb-screenshare-view class="productdb-el" style="height: 480px">
  <productdb-name-tag class="productdb-el">
    <productdb-audio-visualizer class="productdb-el" slot="start"></productdb-audio-visualizer>
  </productdb-name-tag>
</productdb-screenshare-view>

<script>
  const elements = document.getElementsByClassName('productdb-el');
  for (const el of elements) {
    el.participant = meeting.self;
  }
</script>
```
