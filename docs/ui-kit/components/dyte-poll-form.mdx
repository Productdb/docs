# ProductDB-poll-form

A component that lets you create a poll.

```html
<ProductDB-poll-form id="ProductDB-el"></ProductDB-poll-form>

<script>
  document.getElementById('ProductDB-el').addEventListener('ProductDBCreatePoll', (e) => {
    console.log('create poll', e.detail);
  });
</script>
```
