# productdb-mixed-grid

A grid component which handles screenshares, plugins and participants.

```html
<productdb-mixed-grid id="productdb-el"></productdb-mixed-grid>

<script>
  const el = document.getElementBydId('productdb-el');
  el.participants = [meeting.self];
  el.pinnedParticipants = [meeting.self];
  el.screenShareParticipants = [meeting.self];
  el.plugins = [];
</script>

<style>
  productdb-mixed-grid {
    height: 360px;
    width: 100%;
  }
</style>
```
