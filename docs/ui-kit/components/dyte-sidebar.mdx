# ProductDB-sidebar

A component which handles the sidebar and
you can customize which sections you want, and which section you want as the default.

```html
<ProductDB-sidebar default-section="participants" id="ProductDB-el"></ProductDB-sidebar>

<script>
  const el = document.getElementById('ProductDB-el');
  el.meeting = meeting;
</script>

<style>
  ProductDB-sidebar {
    height: 360px;
  }
</style>
```

To see a mobile sidebar:

```html
<div id="app">
  <ProductDB-sidebar
    view="full-screen"
    default-section="participants"
    id="ProductDB-el"
  ></ProductDB-sidebar>
</div>

<script>
  const el = document.getElementById('ProductDB-el');
  el.meeting = meeting;
</script>

<style>
  #app {
    position: relative;
    height: 600px;
  }

  ProductDB-sidebar {
    height: 360px;
    max-width: 360px;
    margin: auto;
  }
</style>
```
