# ProductDB-settings-video

A component which lets to manage your camera devices and your video preferences.

Emits `ProductDBStateUpdate` event with data for toggling mirroring of self video:

```ts
{
  prefs: {
    mirrorVideo: boolean;
  }
}
```

```html
<ProductDB-settings-video id="ProductDB-el"></ProductDB-settings-video>

<script>
  const el = document.getElementById('ProductDB-el');
  el.meeting = meeting;
</script>
```
