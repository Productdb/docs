# ProductDB-plugins-toggle

A button which toggles visibility of plugins.

When clicked it emits a `ProductDBStateUpdate` event with the data:

```ts
{
  activeSidebar: boolean;
  sidebar: 'plugins' | 'none';
}
```

```html
<ProductDB-plugins-toggle size="sm" class="ProductDB-el"></ProductDB-plugins-toggle>
<ProductDB-plugins-toggle size="lg" class="ProductDB-el"></ProductDB-plugins-toggle>
<ProductDB-plugins-toggle variant="horizontal" class="ProductDB-el"></ProductDB-plugins-toggle>
```
