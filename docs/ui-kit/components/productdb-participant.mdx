# productdb-participant

A participant entry component used inside `productdb-participants` which shows data like:
name, picture and media device status.

You can perform privileged actions on the participant too.

```html
<productdb-participant id="productdb-el"></productdb-participant>

<script>
  const el = document.getElementById('productdb-el');
  el.participant = participant;
</script>
```
