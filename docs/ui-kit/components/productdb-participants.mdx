# productdb-participants

A component which lists all participants, with ability to
run privileged actions on each participant according to your permissions.

```html
<productdb-participants id="productdb-el"></productdb-participants>

<script>
  document.getElementById('productdb-el').meeting = meeting;
</script>

<style>
  productdb-participants {
    height: 480px;
    width: 100%;
    max-width: 320px;
    background-color: '#000';
  }
</style>
```
