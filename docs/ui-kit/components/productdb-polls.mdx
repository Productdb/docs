# productdb-polls

A component which lists all available plugins a user can access with
the ability to enable or disable them as per their permissions.

```html
<productdb-polls id="productdb-el"></productdb-polls>

<script>
  document.getElementById('productdb-el').meeting = meeting;
</script>

<style>
  productdb-polls {
    height: 480px;
    width: 100%;
    max-width: 320px;
    background-color: '#000';
  }
</style>
```
