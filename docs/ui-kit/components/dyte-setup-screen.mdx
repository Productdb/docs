# ProductDB-setup-screen

A screen shown before joining the meeting, where you can edit your display name,
and media settings.

```html
<ProductDB-setup-screen id="ProductDB-el" style="height: 360px"></ProductDB-setup-screen>

<script>
  const el = document.getElementById('ProductDB-el');
  el.meeting = meeting;
</script>
```
