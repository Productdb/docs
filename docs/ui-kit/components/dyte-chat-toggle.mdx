# ProductDB-chat-toggle

A button which toggles visibility of chat.

You need to pass the `meeting` object to it to see the unread messages count badge.

When clicked it emits a `ProductDBStateUpdate` event with the data:

```ts
{
  activeSidebar: boolean;
  sidebar: 'chat' | 'none';
}
```

```html
<ProductDB-chat-toggle size="sm" class="ProductDB-el"></ProductDB-chat-toggle>
<ProductDB-chat-toggle size="lg" class="ProductDB-el"></ProductDB-chat-toggle>
<ProductDB-chat-toggle variant="horizontal" class="ProductDB-el"></ProductDB-chat-toggle>

<script>
  const elements = document.getElementsByClassName('ProductDB-el');
  for (const el of elements) {
    el.meeting = meeting;
  }
</script>
```
