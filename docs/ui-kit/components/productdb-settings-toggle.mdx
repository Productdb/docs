# productdb-settings-toggle

A button which toggles visibility of chat.

When clicked it emits a `dyteStateUpdate` event with the data:

```ts
{
  activeSettings: boolean;
}
```

```html
<productdb-settings-toggle size="sm"></productdb-settings-toggle>
<productdb-settings-toggle size="lg"></productdb-settings-toggle>
<productdb-settings-toggle variant="horizontal"></productdb-settings-toggle>
```
