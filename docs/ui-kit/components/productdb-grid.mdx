# productdb-grid

The main grid component which abstracts all the grid handling logic and renders it for you.

```html
<productdb-grid id="productdb-el"></productdb-grid>

<script>
  const el = document.getElementById('productdb-el');
  el.meeting = meeting;
</script>

<style>
  productdb-grid {
    height: 360px;
  }
</style>
```
