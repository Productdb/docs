# ProductDB-fullscreen-toggle

A button which toggles full screen mode for any
existing `ProductDB-meeting` component in the DOM.

```html
<ProductDB-fullscreen-toggle></ProductDB-fullscreen-toggle>
<ProductDB-fullscreen-toggle variant="horizontal"></ProductDB-fullscreen-toggle>
```
