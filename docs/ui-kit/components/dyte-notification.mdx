# ProductDB-notification

A component which shows a notification.

You need to remove the element after you receive the
`ProductDBNotificationDismiss` event.

```html
<ProductDB-notification id="ProductDB-el"></ProductDB-notification>

<script>
  const el = document.getElementById('ProductDB-el');

  el.addEventListener('ProductDBNotificationDismiss', (e) => {
    e.target.remove();
  });

  el.notification = {
    id: 'your-id',
    message: 'Vaibhav says hi!',
    image: 'https://github.com/vaibhavshn.png',
    button: {
      text: 'Say Hi back',
      variant: 'ghost',
      onClick: () => alert('Hey'),
    },
  };
</script>
```
