# productdb-switch

A switch component which follows ProductDB's Design System.

```html
<productdb-switch id="productdb-el"></productdb-switch>

<script>
  const el = document.getElementById('productdb-el');
  el.addEventListener('dyteChange', (e) => {
    alert('New switch value:', e.detail);
  });
</script>
```
