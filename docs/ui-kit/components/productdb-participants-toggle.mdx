# productdb-participants-toggle

A button which toggles visibility of participants.

When clicked it emits a `dyteStateUpdate` event with the data:

```ts
{
  activeSidebar: boolean;
  sidebar: 'participants' | 'none';
}
```

```html
<productdb-participants-toggle size="sm" class="productdb-el"></productdb-participants-toggle>
<productdb-participants-toggle size="lg" class="productdb-el"></productdb-participants-toggle>
<productdb-participants-toggle
  variant="horizontal"
  class="productdb-el"
></productdb-participants-toggle>
```
