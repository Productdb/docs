# UIKit components for Polls

Using ProductDB's UIKit you can simplify the process of adding polls in your meetings!

### Prerequisites

1. Install Core SDK for your framework
2. Install UIKit for your framework

## ProductDB Poll Form : Create a new poll

If a user has the right set of permissions, the first thing they woulds want to do is create a poll for the meeting. The easiest way to get started is to grab the `ProductDB Poll Form` which allows you to create a poll. Check out the implementation based on the framework you are using:

<Tabs groupId="framework">
<TabItem value="web-components" label="Web Components">

```jsx
  <productdb-poll-form id="productdb-el"></productdb-poll-form>
    <script>
      document.getElementById('productdb-el').addEventListener('dyteCreatePoll', (e) => {
      console.log('create poll', e.detail)
      });
    </script>
```

</TabItem>
<TabItem value="react" label="React">

```jsx
<ProductDBPollForm onProductDBCreatePoll={(e) => console.log('create poll', e.detail)} />
```

You can control what happens after a poll is creating by using `onProductDBCreatePoll` prop.

</TabItem>


## ProductDB Polls : View existing polls

After a user is done creating a poll, they can see all the polls available in a meeting using `ProductDB Polls` and enable/disable the polls.

<Tabs groupId="framework">
<TabItem value="web-components" label="Web Components">

```jsx
      <productdb-polls id="productdb-el"></productdb-polls>

      <script>
        document.getElementById('productdb-el').meeting = meeting;
      </script>

      <style>
        productdb-polls {
          height: 480px;
          width: 100%;
          max-width: 320px;
          background-color: '#000';
        }
      </style>
```

</TabItem>

## ProductDB Poll: View poll

The `ProductDB Poll` component lets the users view a given poll. It requires a poll object to render the poll.

<Tabs groupId="framework">
<TabItem value="web-components" label="Web Components">

```jsx
      <productdb-poll id="productdb-el"></productdb-poll>

      <script>
        const el = document.getElementById('productdb-el');

        el.addEventListener('dyteVotePoll', (e) => {
          console.log('Voted', e.detail);
        });

        el.poll = {
          id: 'poll-id',
          question: 'Have you started using productdb yet?',
          options: [
            {
              text: 'Yes',
              votes: [{ id: 'vaibhavs-user-id', name: 'Vaibhav' }],
              count: 0,
            },
            {
              text: 'Nope',
              votes: [],
              count: 0,
            },
          ],
          anonymous: false,
          hideVotes: false,
          createdBy: 'Vaibhav',
          createdByUserId: 'vaibhavs-user-id',
          voted: [],
        };
      </script>
```

</TabItem>

## ProductDB Polls Toggle

Using the `ProductDB Polls Toggle`, one can easily change the visibility of a poll on the screen. It requires the user's meeting object to see the unread polls count badge on the component.

<Tabs groupId="framework">
<TabItem value="web-components" label="Web Components">

```jsx
        <productdb-polls-toggle variant="horizontal" class="productdb-el"></productdb-polls-toggle>
        <script>
          const elements = document.getElementsByClassName('productdb-el');
          for (const el of elements) {
            el.meeting = meeting;
          }
        </script>
```

</TabItem>
<TabItem value="react" label="React">

```jsx
<ProductDBPollsToggle variant="horizontal" size="sm" meeting={meeting} />
```

</TabItem>
<TabItem value="angular" label="Angular">

```jsx
      <productdb-polls-toggle variant="horizontal" class="productdb-el"></productdb-polls-toggle>

      <script>
        const elements = document.getElementsByClassName('productdb-el');
        for (const el of elements) {
```
