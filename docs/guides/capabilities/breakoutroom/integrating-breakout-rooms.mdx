## Step 2: Initialize the SDK

|             |                                                                                                                                                                                   |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `authToken` | After you've created the meeting, add each participant to the meeting using the [Add Participant API](/api#/operations/add_participant). The API response contains the authToken. |

```js
let meeting = await ProductDBClient.init({
  authToken,
});

// add additional event handler for breakout rooms
meeting.connectedMeetings.on('meetingChanged', (newMeeting) => {
  meeting = newMeeting;
});
```

  <TabItem value="Angular">

```js
class AppComponent {
  title = 'MyProject';
  @ViewChild('myid') meetingComponent: ProductDBMeeting;
  ProductDBMeeting: ProductDBClient;

  async ngAfterViewInit() {
    const meeting = await ProductDBClient.init({
      authToken: '<auth-token>',
    });
    meeting.join();
    this.ProductDBMeeting = meeting;
    if (this.meetingComponent) this.meetingComponent.meeting = meeting;
  }
}
```

For detailed guide, check out - https://productdb.dev/angular-ui-kit/quickstart
