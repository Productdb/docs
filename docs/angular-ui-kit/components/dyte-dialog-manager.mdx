# ProductDB-dialog-manager

A component which handles all dialog elements in a component such as:

- ProductDB-settings
- ProductDB-leave-meeting
- ProductDB-permissions-message
- ProductDB-image-viewer
- ProductDB-remote-access-manager
- ProductDB-breakout-rooms-manager

This components depends on the values from `states` object.
