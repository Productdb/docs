# ProductDBPluginsScreen

A widget that provides a well-built plugins screen using molecular components. This widget integrates with the ProductDB UI Kit and return our plugins page as a widget.

:::info

[`ProductDBProvider`](./ProductDB-provider.mdx) needs to be added above `ProductDBPluginsScreen` which will manage ProductDB client object for polls screen by itself.

:::

## Usage

```dart
ProductDBPluginsScreen()
```
