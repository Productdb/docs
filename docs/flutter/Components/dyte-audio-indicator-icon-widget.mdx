# ProductDBAudioIndicatorIconWidget

A widget that displays audio status of a participant in ProductDB meeting.

:::info

[`ProductDBProvider`](./ProductDB-provider.mdx) needs to be added above `ProductDBAudioIndicatorIconWidget` which will manage ProductDB client object for polls screen by itself.

:::

## Usage

```dart
ProductDBAudioIndicatorIconWidget(
  participant: participant
);

```

Here, `participant` is ProductDB participant object of type `ProductDBJoinedMeetingParticipant` of whom you want to show audio status.
