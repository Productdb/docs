# ProductDBParticipantsIconWidget

This widget is designed to integrate a participants icon within a ProductDB-based Flutter application. It facilitates navigation to the `ProductDBParticipantPage` when pressed, serving as a user interface element for accessing the participants list of a ProductDB session.

:::info

[`ProductDBProvider`](./ProductDB-provider.mdx) needs to be added above `ProductDBParticipantsIconWidget` which will manage ProductDB client object for polls screen by itself.

:::

## Usage

To use the `ProductDBParticipantsIconWidget` in a Flutter application, instantiate it directly in your widget tree:

```dart
ProductDBParticipantsIconWidget()
```

## Details

Upon interacting with the widget, the following actions occur:

- The application navigates to the `ProductDBParticipantPage`, which displays the list of participants in the current session.
