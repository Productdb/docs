# ProductDBShowPollsViewController

## Overview
A view controller used to show list of existing polls and a button to create polls

### Topics:

### Creating a show polls screen. 

```swift
 init(meeting: ProductDBMobileClient)
```
#### Parameters:
**meeting:** Current ongoing meeting object. 


### To present ProductDBShowPollsViewController do in iOS as follows:

```swift
let controller = ProductDBShowPollsViewController(dyteMobileClient: self.dyteMobileClient)
self.present(controller, animated: true)
```
