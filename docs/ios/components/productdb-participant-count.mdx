# ProductDBParticipantCountView
## Overview
Subclass of [UILabel](https://developer.apple.com/documentation/uikit/uilabel) which shows count of total joined participants in a meeting.

### Topics:
### Creating a ProductDBParticipantCountView. 

```swift
 init(meeting: ProductDBMobileClient, appearance: ProductDBTextAppearance = AppTheme.shared.participantCountAppearance) {
```
#### Parameters:
*Required*  
**meeting:** Current ongoing meeting object.     
*Optional:*    
**appearance:** Any type which conforms to protocol "ProductDBTextAppearance"

For example you can use this as below. 

```swift
public lazy var lblSubtitle: ProductDBParticipantCountView = {
  let label = ProductDBParticipantCountView(meeting: self.meeting)
  label.textAlignment = .left
  return label
}()
```
