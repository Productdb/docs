# ProductDBControlBarButton
## Overview
Subclass of [UIButton](https://developer.apple.com/documentation/uikit/uibutton)  
A skeleton component used for composing ProductDB buttons which are mostly add inside ProductDBTabbarBar
This is a base class for component likes [ProductDBAudioButtonControlBar](/ios/components/productdb-mic-toggle) [ProductDBMoreButtonControlBar](/ios/components/productdb-more-toggle)

### Creating button 
```swift 
init(image: ProductDBImage, title: String = "", appearance: ProductDBControlBarButtonAppearance = ProductDBControlBarButtonAppearanceModel()) 
```
#### Parameters
*Required*  
**image:** Instance of type `ProductDBImage` to be passed
*Optional*
**title:** title to be shown below the ProductDBImage. Default value of this is Empty Strings.
**appearance:** Any type that follows `ProductDBControlBarButtonAppearance`
