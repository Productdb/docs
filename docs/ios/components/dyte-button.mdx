# ProductDBButton
## Overview
A button that follows ProductDB's Design System.

### Topics:

### Creating a button.

```swift
public init(style: Style = .solid, ProductDBButtonState: States = .active, size: Size = .large, appearance: ProductDBButtonAppearance = ProductDBButtonAppearanceModel()) 
```
Creates ProductDBButton for the **Style** **States** **Size** you passed.

#### Parameters:
*Optional*  
**style:** passed any type of Style. Default value of style is `solid`.   
**ProductDBButtonState:** passed any type of States. Default value of states is `active`.   
**size:**  passed any type of Size. Default value of size is `large`.  
**appearance:** Any type which conforms to protocol `ProductDBButtonAppearance`

### Style
```swift
public enum Style {
  case solid
  case line
  case iconLeftLable(icon: ProductDBImage)
  case iconRightLable(icon: ProductDBImage)
  case text
  case textIconLeft(icon: ProductDBImage)
  case textIconRight(icon: ProductDBImage)
  case iconOnly(icon: ProductDBImage)
  case splitButton
}
```

### States
```swift
public enum States
{
  case active
  case disabled
  case hover
  case focus
  case pressed
}
```

### Size
```swift
public enum Size {
  case small
  case medium
  case large 
}
```
For example we can use ProductDBButton as follows
```swift
let ProductDBImage = ProductDBImage(image:ImageProvider.image(named: "icon_down_arrow"))
let downloadButton: ProductDBButton = {
    let button = ProductDBButton(style: .iconOnly(icon: ProductDBImage),
                  ProductDBButtonState: .active)
    button.backgroundColor = ProductDBSharedTokenColor.background.shade800
    button.isUserInteractionEnabled = false
    button.tintColor = .white
    // Set additional button properties if needed
    button.selectedStateTintColor = DesignLibrary.shared.color.status.danger
    return button
}()
```
