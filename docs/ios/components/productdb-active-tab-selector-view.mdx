# ProductDBActiveTabSelectorView
## Overview
A view which show list of button of Type [ProductDBPluginScreenShareTabButton](/ios/components/productdb-pluginScreenShareTab-button) shown
 horizontally.

<img src="/static/ios/ios_active_HorizontalView.png" alt="Example Image" width="400" height="200"/>


```swift
var buttons: [ProductDBPluginScreenShareTabButton]
```
Array of buttons of Type [ProductDBPluginScreenShareTabButton](/ios/components/productdb-pluginScreenShareTab-button) which are shown inside this list. 


```swift
func scrollToVisible(button: ProductDBPluginScreenShareTabButton)
```
When there are buttons more than width of this view, Then use this method to scroll to button.
#### Parameters:
**button:** button which should must be present inside `var buttons: [ProductDBPluginScreenShareTabButton]`


```swift
func setAndDisplayButtons(_ buttons: [ProductDBPluginScreenShareTabButton]) 
```
This method is used to update buttons on this view. Setting buttons will reset buttons on this view.
#### Parameters:
**buttons:** Array of [ProductDBPluginScreenShareTabButton](/ios/components/productdb-pluginScreenShareTab-button)
 to be shown on `ProductDBActiveTabSelectorView`
