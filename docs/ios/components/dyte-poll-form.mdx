# ProductDBCreatePollsViewController
## Overview
A view controller used to create and publish a poll.

### Topics:
### Creating a ProductDBCreatePollsViewController. 

```swift
 init(meeting: ProductDBMobileClient)
```
#### Parameters:
**meeting:** Current ongoing meeting object. 

### To show ProductDBCreatePollsViewController do in iOS as follows:

```swift
let controller = ProductDBCreatePollsViewController(ProductDBMobileClient: ProductDBMobileClient)
self.present(controller, animated: true)
```
