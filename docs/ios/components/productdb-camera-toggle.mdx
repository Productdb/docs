# ProductDBSwitchCameraButtonControlBar
## Overview
This button is use to implement the action for toggling the Camera from front to back and vice-versa.


### Topics:

### Creating a Camera toggles button. 

```swift
 init(meeting: ProductDBMobileClient)
```
#### Parameters:
**meeting:** Current ongoing meeting object. 


### Override the onClick action. 
```swift
func onClick(button: ProductDBControlBarButton)
```
You can override this method inside the subclass of `ProductDBSwitchCameraButtonControlBar` to customize the functionality when user presses this button.
