# ProductDBClockView
## Overview
This view is used to show elapsed time for the meeting in the format Hours:Minute:Seconds.


### Topics:

### Creating a clock view

```swift
 init(meeting: ProductDBMobileClient, appearance: ProductDBTextAppearance = AppTheme.shared.clockViewAppearance) 
```
Creates a ProductDBClockView for meeting passed. 

#### Parameters:  
*Required*  
**meeting:** Current ongoing meeting object.  
*Optional*  
**appearance:** Any type which conforms to protocol "ProductDBTextAppearance"

```swift
private lazy var clockView: ProductDBClockView = {
    let label = ProductDBClockView(meeting: self.meeting)
    label.textAlignment = .left
    return label
}()
```
