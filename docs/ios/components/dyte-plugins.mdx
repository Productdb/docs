# ProductDBPluginViewController
## Overview
A viewcontroller which lists all available plugins from their preset, and ability to
enable or disable plugins.


### Topics:
### Creating a ProductDBPluginViewController. 

```swift
init(plugins: [ProductDBPlugin])
```
#### Parameters:
**meeting:** Current ongoing meeting object. 



```swift
let controller = ProductDBPluginViewController(plugins: ProductDBMobileClient.plugins.all)
let navigationController = UINavigationController(rootViewController: controller)
navigationController.modalPresentationStyle = .fullScreen
present(navigationController, animated: true, completion: nil)
```
