# ProductDBPluginsView

## Overview

A view which show individual (plugin/screen share) view. On top of this view we are showing instance of
[ProductDBActiveTabSelectorView](/ios/components/ProductDB-active-tab-selector-view).
In case of Mutliple(Count more than 1) plugins or Screen share, We are showing `ProductDBActiveTabSelectorView`
else it will be hidden.

## Topics:

### Create plugins view

```swift
init(videoPeerViewModel: VideoPeerViewModel)
```

#### Parameters

**videoPeerViewModel** Instance of type [VideoPeerViewModel](/ios/helper-classes/ProductDB-plugin-view-model)

### Adding plugins tab button

```swift
func setButtons(buttons: [ProductDBPluginScreenShareTabButton],  selectedIndex: Int?, clickAction:@escaping(ProductDBPluginScreenShareTabButton, Bool)->Void)
```

#### Parameters

**buttons:** Array of ProductDBPluginScreenShareTabButton to be shown inside [ProductDBActiveTabSelectorView](/ios/components/ProductDB-active-tab-selector-view)  
**selectedIndex:** Index of plugin button which needs to be selected from `buttons: [ProductDBPluginScreenShareTabButton]`  
**clickAction:** Call back of type `(ProductDBPluginScreenShareTabButton, Bool)->Void`.<br/> _First parameter_ is the button selected by user
.<br/> _Second parameter_ is bool that tells whether button tapped is plugin or either screen share button. `true` is plugin/screen share button clicked.
