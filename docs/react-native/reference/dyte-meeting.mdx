# ProductDBMeeting (React component)

ProductDBMeeting is a React component that can be included in your React application
to show the ProductDB Meeting UI.

## Props

| Prop name     | Data type                      | Required | Use                                                                                                                                                       |
| ------------- | ------------------------------ | -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| onInit        | function(meeting: ProductDBMeeting) | Yes      | On successful meeting creation, returns the meeting to the callback function provided (see [Meeting object](./meeting) for more details)                  |
| onError       | function(error: Error)         | No       | On error during meeting creation, returns the relevant error to the callback function provided (see [Error handling](./../usage#errors) for more details) |
| clientId      | string                         | Yes      | The `orgId` obtained from ProductDB                                                                                                                            |
| meetingConfig | ConnectionConfig               | Yes      | Meeting config object used to start the meeting (see [Connection config](./connection-config) for more details)                                           |
| uiConfig      | ProductDBUIConfig                   | No       | UI configuration options (see [UI layout](./ProductDB-ui-config) for more details)                                                                             |
