# ProductDBMoreToggle

A button which toggles visibility of a more menu.

When clicked it emits a `ProductDBStateUpdate` event with the data:

```ts
{
  activeMoreMenu: boolean;
}
```

```jsx
<Center>
  <ProductDBMoreToggle size="sm" />
  <ProductDBMoreToggle size="lg" />
  <ProductDBMoreToggle variant="horizontal" size="sm" />
</Center>
```
