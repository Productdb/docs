# ProductDBParticipantTile

A component which plays a participants video and allows for placement of
components like `ProductDB-name-tag`, `ProductDB-audio-visualizer` or any other component.

```jsx
<ProductDBParticipantTile participant={meeting.self}>
  <ProductDBNameTag participant={meeting.self}>
    <ProductDBAudioVisualizer slot="start" />
  </ProductDBNameTag>
</ProductDBParticipantTile>
```

You can change the `name-tag-position` to any of the supported values and change
the placement of audio-visualizer in name-tag as well.

```jsx
<ProductDBParticipantTile participant={meeting.self} nameTagPosition="bottom-center">
  <ProductDBNameTag participant={meeting.self}>
    <ProductDBAudioVisualizer slot="end" />
  </ProductDBNameTag>
</ProductDBParticipantTile>
```

It also has a few variants.

```jsx
<ProductDBParticipantTile
  participant={meeting.self}
  nameTagPosition="bottom-center"
  variant="gradient"
>
  <ProductDBNameTag participant={meeting.self}>
    <ProductDBAudioVisualizer slot="start" />
  </ProductDBNameTag>
</ProductDBParticipantTile>
```
