# ProductDBAvatar

Avatar component which renders a participant's image or their initials.

```jsx
<Row>
  <ProductDBAvatar size="sm" />
  <ProductDBAvatar size="md" participant={meeting.self} />
  <ProductDBAvatar size="lg" />
</Row>
```
