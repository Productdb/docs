# ProductDBPollsToggle

A button which toggles visibility of polls.

You need to pass the `meeting` object to it to see the unread polls count badge.

When clicked it emits a `ProductDBStateUpdate` event with the data:

```ts
{
  activeSidebar: boolean;
  sidebar: 'polls' | 'none';
}
```

```jsx
<Center>
  <ProductDBPollsToggle size="sm" meeting={meeting} />
  <ProductDBPollsToggle size="lg" meeting={meeting} />
  <ProductDBPollsToggle variant="horizontal" size="sm" meeting={meeting} />
</Center>
```
