# ProductDBMenuList

A menu list component.

```jsx
<Row style={{ padding: '52px 0 0 0' }}>
  <ProductDBMenu placement="top">
    <ProductDBButton slot="trigger">Top Menu</ProductDBButton>
    <ProductDBMenuList>
      <ProductDBMenuItem onClick={() => alert('You clicked: alert()')}>
        alert()
      </ProductDBMenuItem>
    </ProductDBMenuList>
  </ProductDBMenu>

  {/* This menu will be placed at top due to less space */}
  <ProductDBMenu placement="bottom">
    <ProductDBButton slot="trigger">Bottom Menu</ProductDBButton>
    <ProductDBMenuList>
      <ProductDBMenuItem onClick={() => alert('You clicked: alert()')}>
        alert()
      </ProductDBMenuItem>
    </ProductDBMenuList>
  </ProductDBMenu>
</Row>
```
