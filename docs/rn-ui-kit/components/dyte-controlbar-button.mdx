# ProductDBControlbarButton

A skeleton component used for composing buttons.

```jsx
<Center>
  <ProductDBControlbarButton
    label="Flight Mode"
    icon={airplaneSVG}
    onClick={() => alert('Flight mode toggled')}
    size="sm"
  />
  <ProductDBControlbarButton
    label="Flight Mode"
    icon={airplaneSVG}
    onClick={() => alert('Flight mode toggled')}
    size="lg"
  />
  <ProductDBControlbarButton
    label="Flight Mode"
    icon={airplaneSVG}
    onClick={() => alert('Flight mode toggled')}
    variant="horizontal"
    size="sm"
  />
</Center>
```
