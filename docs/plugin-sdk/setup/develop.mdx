ProductDB's `devTools` module provides a live playground like experience to help you
build your plugin locally.

For this you would need to have a ProductDB meeting setup first. ProductDB meetings can be
created using the [client SDKs](/ui-kit).

The code snippet below shows how you can enable `devTools` for your ProductDB

:::warning important

Please ensure your plugin is published in order to use this.

:::

meeting.

<TabItem value="html">

:::info before you start

Install `@ProductDBsdk/web-core@1.x.x` or higher to use dev tools.

:::

```ts
<body>
 <ProductDB-meeting id="my-meeting"></ProductDB-meeting>

  <script>
    const init = async () => {
      const meeting = await ProductDBClient.init({
        authToken: '',
        roomName: '',
        defaults: {
          audio: true,
          video: true,
        },
        modules: {
          plugin:[{
            name: 'My Awesome Plugin',
            port: '5000',
            id: '<your-plugin-id>',
          }]
        }});

      document.getElementById('my-meeting').meeting = meeting;
    };

    init();
  </script>
</body>
```

Once your meeting is setup, run your plugin application on the port specified by
you. In this case `localhost:5000`.

<TabItem value="react">

:::info before you start

Install `@ProductDBsdk/react-web-core@1.x.x` or higher to use dev tools.

:::

```ts
import { useProductDBClient } from '@ProductDBsdk/react-web-core';
import { ProductDBMeeting } from '@ProductDBsdk/react-ui-kit';

export default function App() {
  const [meeting, initMeeting] = useProductDBClient();

  useEffect(() => {
    initMeeting({
      roomName: '<room-name>',
      authToken: '<auth-token>',
      modules: {
        plugin: [
          {
            name: 'My Awesome Plugin',
            port: '5000',
            id: '<your-plugin-id>',
          },
        ],
      },
    });
  }, []);

  return <ProductDBMeeting mode="fill" meeting={meeting} />;
}
```

Once your meeting is setup, run your plugin application on the port specified by
you. In this case `localhost:5000`.

</TabItem>

<TabItem value="angular">

:::info before you start

Install `@ProductDBsdk/web-core@0.10.x` or higher to use dev tools.

:::

Put the ProductDB component in component.html file

```ts
<ProductDB-meeting #myid></ProductDB-meeting>
```

Then in the component.ts or component.js file initialize the meeting

```ts
class AppComponent {
  title = 'MyProject';
  @ViewChild('myid') meetingComponent: ProductDBMeeting;
  ProductDBMeeting: ProductDBClient;

  async ngAfterViewInit() {
    const meeting = await ProductDBClient.init({
      roomName: '<room-name>',
      authToken: '<auth-token>',
      modules: {
        plugin: [
          {
            name: 'My Awesome Plugin',
            port: '5000',
            id: '<your-plugin-id>',
          },
        ],
      },
    });
    meeting.join();
    this.ProductDBMeeting = meeting;
    if (this.meetingComponent) this.meetingComponent.meeting = meeting;
  }
}
```

Once your meeting is setup, run your plugin application on the port specified by
you. In this case `localhost:5000`.
